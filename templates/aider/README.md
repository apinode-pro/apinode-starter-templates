# Aider Setup Starter

Set the OpenAI-compatible base URL and model:

```bash
export OPENAI_API_KEY="your_api_key"
export OPENAI_API_BASE="https://apinode.pro"
aider --model openai/gpt-5.5
```

If your Aider version uses a different base URL flag or environment variable, keep the same values:

```text
Base URL: https://apinode.pro
Model:    gpt-5.5
```
